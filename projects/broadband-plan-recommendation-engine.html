Broadband Plan Recommendation Engine

Jun 2024 - Aug 2024 | Business Consulting Project

Challenge & Objective

Internet service providers need data-driven solutions to recommend optimal broadband plans to customers based on regional availability and demographic needs. This project aimed to build an intelligent recommendation system that serves underserved communities and improves customer satisfaction through personalized plan suggestions.

Approach & Methodology

1. Predictive Modeling

Engineered a synthetic dataset combining US Census demographic data and FCC broadband availability databases. Trained a Random Forest model to predict optimal internet speeds based on age, income, and device usage patterns, achieving 95% accuracy in plan recommendations.

2. Geospatial Analysis

Integrated FIPS codes to map county-level broadband providers and service availability. This enabled personalized recommendations for the top 3 service providers in users' regions, improving accessibility for underserved demographics.

3. End-to-End Solution

Built a Streamlit web app for real-time plan filtering and UX demonstration. The interactive interface allows users to input their profile and receive instant recommendations for optimal broadband plans aligned with their needs and budget.

Results & Impact

  • Prediction Accuracy: 95% accuracy in recommending plans based on demographics (age, income, device usage)
  • Geospatial Coverage: County-level provider mapping enabling personalized top 3 provider recommendations
  • Accessibility: Streamlit web app provides real-time filtering for underserved demographics
  • Business Value: Demonstrates scalable solution for ISPs to improve customer targeting and satisfaction

Tech Stack

Python Random Forest Streamlit Geospatial Analysis US Census Data FCC Data Pandas Scikit-learn

Key Learnings

  • Government datasets (Census, FCC) are powerful but require careful data cleaning and validation
  • FIPS codes are essential for reliable geographic data integration
  • Streamlit significantly reduces time-to-market for machine learning applications
  • Demographic factors are strong predictors for broadband plan preferences
