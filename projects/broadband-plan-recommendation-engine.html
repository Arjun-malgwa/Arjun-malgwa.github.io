Broadband Plan Recommendation Engine

Jun 2024 - Aug 2024 | Business Consulting Project

Challenge & Objective

Internet service providers need data-driven solutions to recommend optimal broadband plans to customers based on regional availability and demographic needs. This project aimed to build an intelligent recommendation system that serves underserved communities and improves customer satisfaction through personalized plan suggestions.

Approach & Methodology

  • Data Integration: Processed US Census demographic data and FCC broadband availability databases using FIPS codes for geographic mapping
  • Feature Engineering: Created feature set combining income levels, population density, available ISPs, and plan speeds
  • Model Development: Built Random Forest classifier achieving 95% accuracy in plan predictions
  • Streamlit Dashboard: Developed real-time interactive interface for end-users to filter and compare plans
  • Geospatial Mapping: Visualized coverage gaps and plan availability by zip code

Results & Impact

  • Prediction Accuracy: 95% accuracy in recommending plans matched to user demographics
  • Coverage: Identified solutions for underserved demographic groups (rural/low-income areas)
  • User Experience: Real-time Streamlit interface enables customers to find optimal plans in seconds
  • Scalability: System ready to process millions of customer profiles

Tech Stack

Python Random Forest Streamlit Geospatial Analysis US Census Data FCC Data Pandas Scikit-learn

Key Learnings

  • Government datasets (Census, FCC) are powerful but require careful data cleaning and validation
  • FIPS codes are essential for reliable geographic data integration
  • Streamlit significantly reduces time-to-market for machine learning applications
  • Demographic factors are strong predictors for broadband plan preferences
