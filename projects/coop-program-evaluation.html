Co-Op Program Evaluation Analysis

Sep 2023 - Dec 2023 | Institutional Research

Challenge & Objective

Drexel University's cooperative education program needed comprehensive evaluation to understand program effectiveness and identify improvement opportunities. The project synthesized perspectives from both students and employers through survey analysis and created actionable insights for program enhancement.

Approach & Methodology

1. Program Analysis

Conducted an insightful analysis of Drexel University's Co-Op program, exploring both student and employer perspectives through comprehensive post-co-op surveys designed to capture satisfaction, skill development, and program effectiveness.

2. Data Processing & Analysis

Leveraged Python for robust data processing and analysis, handling survey responses from students and employers. Applied statistical methods to extract meaningful insights about program strengths and improvement opportunities.

3. Visualization & Communication

Built interactive Tableau visualizations to present findings in an accessible format. Created dashboards enabling stakeholders to explore data independently and understand multi-perspective insights about program effectiveness.

Results & Impact

  • Multi-Perspective Insights: Synthesized both student and employer feedback to understand program effectiveness from multiple angles
  • Interactive Dashboards: Built Tableau visualizations enabling stakeholders to explore data and generate insights independently
  • Comprehensive Analysis: Identified key program strengths and improvement opportunities through Python data processing
  • Business Value: Provided actionable recommendations for co-op program enhancement and stakeholder satisfaction

Tech Stack

Python Tableau Data Analysis Survey Analytics Sentiment Analysis Pandas Numpy Statistical Analysis

Key Learnings

  • Survey design significantly impacts data quality and response rates
  • Stakeholder perspectives (students vs employers) often reveal different program strengths
  • Tableau's interactive dashboards empower non-technical stakeholders to explore data
  • Sentiment analysis complements quantitative metrics to understand qualitative feedback
  • Actionable insights require deep understanding of institutional context
