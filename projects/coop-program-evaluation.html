Co-Op Program Evaluation Analysis

Sep 2023 - Dec 2023 | Institutional Research

Challenge & Objective

Drexel University's cooperative education program needed comprehensive evaluation to understand program effectiveness and identify improvement opportunities. The project synthesized perspectives from both students and employers through survey analysis and created actionable insights for program enhancement.

Approach & Methodology

  • Survey Design: Designed and distributed comprehensive post-program surveys to students and employers
  • Data Collection: Gathered 200+ responses covering satisfaction, skill development, and program logistics
  • Analysis: Applied descriptive statistics, correlation analysis, and sentiment analysis on survey responses
  • Visualization: Built interactive Tableau dashboards showing key metrics and insights by department
  • Insights: Identified key program strengths and areas for improvement across stakeholder groups

Results & Impact

  • Comprehensive Report: Delivered detailed analysis with actionable recommendations for program improvement
  • Multi-Perspective Insights: Captured both student and employer satisfaction (85% positive sentiment)
  • Interactive Dashboards: Developed Tableau dashboards enabling stakeholders to explore data independently
  • Program Enhancement: Recommendations implemented led to structural improvements in subsequent cycles

Tech Stack

Python Tableau Data Analysis Survey Analytics Sentiment Analysis Pandas Numpy Statistical Analysis

Key Learnings

  • Survey design significantly impacts data quality and response rates
  • Stakeholder perspectives (students vs employers) often reveal different program strengths
  • Tableau's interactive dashboards empower non-technical stakeholders to explore data
  • Sentiment analysis complements quantitative metrics to understand qualitative feedback
  • Actionable insights require deep understanding of institutional context
